# Application de Gestion de Ferme Laitière

## Présentation

Cette application web a été développée dans le cadre d’un projet de fin d’études (PFE). Elle permet de gérer efficacement les activités d’une ferme laitière grâce à une interface simple, moderne et facile à utiliser.

L’objectif principal de cette application est de faciliter le suivi des animaux, de la production laitière, des dépenses et des revenus afin d’améliorer l’organisation et la gestion quotidienne de la ferme.

## Fonctionnalités

* Gestion des vaches et de leurs informations.
* Suivi de la production laitière.
* Gestion des naissances.
* Gestion des traitements et du suivi sanitaire.
* Gestion des dépenses et des revenus.
* Consultation des statistiques et des rapports.
* Tableau de bord permettant un suivi rapide des activités de la ferme.

## Technologies utilisées

* Python
* Flask
* MySQL
* phpMyAdmin
* HTML
* CSS
* JavaScript
